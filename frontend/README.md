# Frontend — Desafio Técnico MinhaFabrica

Interface administrativa desenvolvida com Next.js, React, TypeScript e Tailwind CSS.

## Como rodar

```bash
npm install
npm run dev
```

Acesse: http://localhost:3000

## Variáveis de ambiente

Crie um arquivo `.env.local` baseado no `.env.example`:

```env
NEXT_PUBLIC_API_URL=http://localhost:5000/api/v1
```

> O README completo do projeto está na raiz do repositório.
